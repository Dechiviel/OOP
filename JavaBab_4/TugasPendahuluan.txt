4. Uraikan pengetahuan anda mengenai array multidimensi!

array multidimensi adalah array yang memiliki lebih dari 2 dimensi, dimana sebuah array dapat berisi array lagi. dalam dari array yang berisi array ditentukan oleh banyaknya dimensi

5. Bagaimana cara mengkopi isi array?

Isi dari suatu array dapat dikopi pada array yang lain dengan memanfaatkan method arraycopy() pada class System. Format penulisannya sebagai berikut :

System.arraycopy(array1, p1, array2, p2, n);
dimana : 
	array1 = array asal/sumber pengkopian
	array2 = array tujuan pengkopian
	p1 = posisi indeks awal pengkopian pada array asal
	p2 = posisi indeks awal pengkopian pada array tujuan
	n = banyaknya elemen array yang akan dikopi

6. Apakah yang dimaksud dengan referensi array. Beri contoh!

referensi array adalah dimana alamat dari sebuah array dapat ditunjuk oleh array lainnya

contoh:
int arrayA[] = {10, 20, 30};
int arrayB[];
arrayB = arrayA (elemen arrayA dapat di akses dan dimodifikasi melalui arrayB)
